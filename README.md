# Proje Özeti

Bu proje, içerik yönetimi ve kullanıcı rolleri odaklı bir haber sitesi api’si geliştirmeyi amaçlamıştır. Api, farklı kullanıcı rolleri (admin, haber editörü, kullanıcı) ile içerik paylaşımı ve yönetimini sağlayacaktır.


# Teknik Özellikler

Django Rest Framework, Postgresql veritabanı, JWT authentication, sorgu optimizasyonu, Swagger ile api dokümentasyonu.
